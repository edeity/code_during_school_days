## 简介
大一暑假折腾的小游戏,纯粹的鼠标操作

来一个很挫的游戏界面：

## 操作
+ 左右滑动：改变坦克方向
+ 滑轮滚动：切换武器
+ 长按左键：移动
+ 长按右键：射击
